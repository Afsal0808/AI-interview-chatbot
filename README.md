# AI Interview Bot (Voice + NLP)

## 📌 Description
This project is an AI-powered interview system that asks questions,
records user answers via voice, converts them into text, and evaluates performance.

## 🚀 Features
- Voice-based answers
- Speech-to-text conversion
- NLP-based evaluation
- Feedback system

## 🛠️ Technologies Used
- Python
- Streamlit
- SpeechRecognition
- NLP (spaCy / NLTK)
- Machine Learning

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py

## 📊 Future Enhancements
- Emotion detection
- Resume-based questions
- AI scoring using BERT
